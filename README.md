# ASCII LENS

ASCII LENS is a Vite + React app for turning images into stylized ASCII artwork.

## Public URL

https://asciilens.vercel.app/

## Usage Audit

AsciiLens can record anonymous usage events and expose a token-protected summary API for agents through Vercel and Supabase. See [docs/audit.md](docs/audit.md).

## Development

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```
